# Payments dashboard

## Dashboard Overview <a href="#dashboard-overview" id="dashboard-overview"></a>

### **Main Navigation**

The dashboard is organized into primary sections accessible from the left sidebar:

* **Overview**: Real-time transaction summary and key metrics
* **Transactions**: Detailed transaction history and search
* **Analytics**: Access to Reporting and Usage Insights features
* **Settings**: Account configuration and team management
* **Enterprise**: Advanced features like Smart Routing and Smart Forecasting (Enterprise accounts only)

### **Quick Stats Bar**

At the top of every page, you'll find:

* Today's transaction volume and count
* Current success rate percentage
* Active payment methods
* System status indicator

## Main Dashboard Sections <a href="#main-dashboard-sections" id="main-dashboard-sections"></a>

### **Overview Page**

Your landing page shows:

**Transaction Summary**: Today's volume, successful payments, and failed attempts with comparison to yesterday&#x20;

**Recent Activity**: Latest transactions with status and amounts&#x20;

**Performance Metrics**: Success rates by payment method and processor&#x20;

**Quick Actions**: Buttons to create test transactions, view reports, or access settings

### **Transactions Section**

**Transaction Search**: Find specific payments using filters for date, amount, status, or customer information&#x20;

**Transaction Details**: Click any transaction to view complete payment information, processor response, and customer data&#x20;

**Bulk Actions**: Export transaction data or perform batch operations on selected payments&#x20;

**Dispute Management**: View and respond to chargebacks and payment disputes

### **Analytics Section**

**Reporting**: Access to your configured reports, report templates, and scheduling options&#x20;

**Usage Insights**: View team activity, API usage statistics, and feature adoption metrics&#x20;

**Performance Trends**: Charts showing transaction volumes, success rates, and revenue over time

**Custom Dashboards**: Create personalized views with the metrics most important to your role

### **Settings Areas**

#### **User Settings**

Access your personal settings via the user menu (top right corner):

**Profile Information**: Update your name, email, and contact preferences&#x20;

**Notification Preferences**: Choose which alerts and reports you receive via email&#x20;

**API Keys**: Generate and manage your personal API credentials&#x20;

**Two-Factor Authentication**: Enable additional security for your account&#x20;

**Session Management**: View active sessions and sign out of other devices

#### **Organization Settings**

Found under Settings > Organization (admin access required):

**Company Profile**: Business name, address, and tax information&#x20;

**Team Management**: Invite users, assign roles, and manage permissions&#x20;

**Payment Configuration**: Set up processors, enable payment methods, and configure routing&#x20;

**Billing Settings**: View invoices, update payment methods, and manage subscription&#x20;

**Security Settings**: Enforce two-factor authentication, set password policies, and review audit logs

#### **Enterprise Features**

For Enterprise accounts, additional sections appear:

**Smart Routing**: Configure intelligent payment routing rules and monitor performance&#x20;

**Smart Forecasting**: Access prediction models and scenario planning tools&#x20;

**Advanced Reporting**: Create custom report templates and set up automated distribution&#x20;

**Compliance Tools**: Generate regulatory reports and manage audit documentation

## Navigation Tips <a href="#navigation-tips" id="navigation-tips"></a>

### **Quick Search**

Use the search bar at the top of the dashboard to quickly find:

* Specific transactions by ID or customer email
* Team members and their contact information
* Documentation and help articles
* Feature settings and configuration pages

### **Breadcrumb Navigation**

The breadcrumb trail below the main navigation shows your current location and allows quick navigation back to parent sections.

### **Keyboard Shortcuts**

* `Ctrl/Cmd + K`: Open quick search
* `Ctrl/Cmd + /`: View keyboard shortcuts help
* `G + T`: Go to transactions
* `G + A`: Go to analytics
* `G + S`: Go to settings

## Customization Options <a href="#customization-options" id="customization-options"></a>

### **Dashboard Layout**

**Widget Configuration**: Add, remove, or rearrange dashboard widgets based on your needs&#x20;

**Metric Selection**: Choose which KPIs appear in your quick stats bar&#x20;

**Default Views**: Set your preferred landing page and transaction list filters&#x20;

**Theme Settings**: Switch between light and dark mode (user preference)

### **Role-Based Views**

The dashboard automatically adapts based on your role:

**Admin Users**: See all sections including organization settings and team management&#x20;

**Finance Users**: Focus on reporting, reconciliation, and billing information&#x20;

**Operations Users**: Emphasize transaction monitoring and customer support tools&#x20;

**Developers**: Highlight API documentation, testing tools, and integration guides
